# NodeBloom 기획서

> **노드가 피다, 지식이 자라다.**
> AI가 만드는 스킬트리, 게임처럼 배우는 학습. 심사위원이 이 문서 하나로 프로젝트를 완전히 이해할 수 있도록 전체 기획을 정리했습니다.

---

## 1. 프로젝트 비전

### 1.1 한 줄 소개
**NodeBloom**은 교사가 수업 자료를 업로드하면 AI가 자동으로 학습 지도(스킬트리)를 만들고, 학생이 퀴즈를 풀어 노드를 하나씩 언락하며 꽃이 피듯 지식을 확장하는 **4자 연동 차세대 AI 교육 플랫폼**입니다.

### 1.2 슬로건과 컨셉
> **"노드가 피다, 지식이 자라다"**

- **노드**: 학습의 최소 단위 (하나의 개념/주제)
- **피다**: 학생이 퀴즈를 풀어 노드를 언락하는 순간
- **자라다**: 언락된 노드가 다음 노드를 열고, 전체 지식이 유기적으로 확장됨
- **브랜드 컬러**: `#6366F1` (인디고 · 지식) · `#A855F7` (연보라 · 꽃잎) · `#10B981` (초록 · 성장)
- **로고**: 원형 노드 + 사방으로 피어나는 꽃잎 4장 + 성장의 잎 + 중심 노드 코어

### 1.3 핵심 가치 3가지
| 가치 | 설명 |
|---|---|
| 🌳 **지식의 시각적 연결** | 선수지식과 응용을 한눈에 보는 스킬트리로 학습의 구조를 명확히 함 |
| 🎮 **게임 같은 몰입** | XP, 레벨, 배지, 스트릭, 인증서로 학습을 재미있게 지속 |
| 👥 **4자의 유기적 연동** | 교사-학생-학부모-운영자가 하나의 학습 여정에서 서로를 돕는 구조 |

## 2. 해결하는 문제

### 2.1 문제 ① — 학습이 지도 없이 진행된다
**현실**: 교과서와 시험지만 있는 공부. 학생은 "왜 지금 이걸 배우는지" 모른 채 순서대로 외운다.
**해결**: PDF 한 장이 **시각적 스킬트리**로 바뀐다. 학생은 전체 지도를 보며 "이걸 끝내면 저게 열린다"는 맥락을 가진다.

### 2.2 문제 ② — 교사의 수업 준비가 과중하다
**현실**: 수업 자료, 퀴즈, 학습지, 복습 문제를 모두 교사가 손으로 만든다.
**해결**: **16종 AI 통합 기능**이 스킬트리·퀴즈·학습지·복습 카드·주간 브리핑까지 자동 생성. 교사는 검수와 소통에 집중.

### 2.3 문제 ③ — 학부모는 자녀의 학습을 전혀 모른다
**현실**: 시험 점수만 보고 판단, 평상시의 노력과 어려움은 보이지 않는다.
**해결**: 자녀의 6자리 코드로 **학부모 대시보드** 연결. 실시간 진도, 감정 상태, AI 주간 리포트로 **이해 기반의 응원**이 가능해진다.

## 3. 타겟 사용자 (4자 역할)

### 3.1 👨‍🏫 교사 (Teacher)
**핵심 플로우**:
1. 스쿨 초대 코드 입력 → 가입
2. 클래스 생성 → 학생 수강신청 승인
3. PDF 업로드 → AI가 스킬트리 자동 생성
4. D3 그래프에서 노드/엣지 편집 → 게시
5. 학생 대시보드로 **감정·위험·진도·병목** 모니터링
6. 수업 녹음 업로드 → Whisper 전사 → Claude 요약 + 복습 퀴즈 생성
7. 학부모 리포트 AI 자동 생성 → 공유

**해결하는 고통**: 수업 자료 작성, 퀴즈 출제, 학부모 소통 모두 AI가 보조

### 3.2 👩‍🎓 학생 (Student)
**핵심 플로우**:
1. 클래스 코드로 수강신청 → 승인 대기 → 학습 시작
2. 학습 스타일 진단 (시각형/텍스트형/실습형)
3. 대시보드: XP · 레벨 · 스트릭 · 오늘의 미션 · AI 주간 플랜
4. 스킬트리 탐험: 노드 클릭 → 학습 문서 읽기 → 퀴즈 풀이
5. 오답 시 AI 튜터에게 소크라틱 질문 (정답 직접 제공 X)
6. 노드 완료 → 플래시카드 5장 자동 생성 → 복습
7. 스킬트리 100% 완료 → **수료 인증서 자동 발급**

**해결하는 고통**: 학습 동기 부족, 진도 파악 어려움, 약점 무자각

### 3.3 👨‍👩‍👧 학부모 (Parent)
**핵심 플로우**:
1. 자녀가 대시보드에서 **6자리 초대 코드** 생성 → 전달
2. 학부모는 `/parent/link`에서 코드 입력 → 자동 연결
3. `/parent` 대시보드에서 자녀의 학습 현황을 한눈에 확인
4. AI가 만든 **주간 리포트**로 강점·개선점 파악
5. 자녀의 감정 상태·위험 신호 조기 발견 → 대화로 연결

**해결하는 고통**: 자녀의 진짜 학습 상태를 모름, 불안 기반의 잔소리

### 3.4 🛡️ 운영자 (Admin)
**핵심 플로우**:
1. 스쿨 생성 → 교사/학생 초대 코드 자동 발급
2. 교사 승인 + 학생 관리
3. 전체 통계: 수업 참여율, 평균 진도, 위험군 학생 수
4. **AI 병목 분석**: 어느 노드에서 학생들이 막히는지
5. 학교 단위 공지사항 발송
6. 1:1 메신저로 교사/학생과 소통

**해결하는 고통**: 학교/학원 운영의 데이터 부재, 교사 업무 효율 파악 불가

| 10 | **HTML 학습지 생성** | 인쇄 가능한 학습지 HTML을 노드별 자동 생성, 교사 스타일 학습 |

## 6. 게이미피케이션 설계

### 6.1 성장 엔진 (XP + 레벨)
- XP 획득원: **노드 완료 +30**, **퀴즈 만점 +25**, **미션 완료 +20~35**
- 레벨 = `floor(xp / 100) + 1`
- 프로필 카드에 Lv.N + 진행 바 표시

### 6.2 일일 미션 5유형
1. 노드 1개 잠금해제하기 (+30 XP)
2. 퀴즈 3개 풀기 (+25 XP)
3. AI 튜터에게 1번 질문하기 (+20 XP)
4. 복습 노드 1개 다시 풀기 (+25 XP)
5. 30분 학습하기 (+35 XP)

매일 랜덤 3개 자동 생성, 진도 자동 추적.

### 6.3 업적/배지 36종
- `first_unlock` 🔓 첫 노드 잠금해제
- `five_unlocks` ⚡ 5개 노드 잠금해제
- `ten_unlocks` 🏃 10개 노드 잠금해제
- `perfect_quiz` 💯 퀴즈 만점
- `perfect_streak` 🔥 7일 스트릭
- ... (총 10종)

### 6.4 학습 스트릭
연속 학습일 기록 → 동기부여. `streak_days` + `last_study_date` 자동 관리.

### 6.5 AI 플래시카드
노드 완료 시 Claude가 5장 질문-답변 카드 자동 생성. "알겠어요/다시볼게요" 선택으로 복습 추적.

### 6.6 수료 인증서
스킬트리 100% 완료 시 자동 발급. HTML 템플릿으로 인쇄 가능 (A4, NodeBloom 로고 + SEAL + 완료 통계).

### 6.7 적응형 복습 (에빙하우스)
- 기본: 1일 → 3일 → 7일 → 14일 → 30일
- 정답률 ≥ 80%: 간격 2배
- 60-79%: 유지
- < 60%: 간격 0.5배 (자주 복습)

## 8. 데이터 모델 요약

### 8.1 핵심 엔티티
```
auth.users (Supabase)
    │
    ▼
profiles (id, email, name, role, xp, streak_days, learning_style)
    │
    ├─────► schools (teacher_code, student_code)
    │           │
    │           ▼
    │       classes (teacher_id)
    │           │
    │           ▼
    │       class_enrollments (student_id, status)
    │
    └─────► skill_trees (class_id, created_by, subject_hint)
                │
                ▼
            nodes (title, difficulty, learning_content)
                │
                ├─► node_edges (source, target)
                │
                ├─► quizzes → quiz_attempts
                │
                ├─► flashcards → flashcard_reviews
                │
                └─► student_progress (status: locked/available/completed)
                        │
                        ▼
                    certificates (스킬트리 100% 완료 시)
```

- **게이미피케이션 (7)**: daily_missions, achievements, user_achievements, review_reminders, flashcards, flashcard_reviews, certificates
- **AI 캐시 (5)**: emotion_reports, weekly_plans, weekly_briefings, tutor_conversations, lesson_recordings
- **학부모 (2)**: parent_invite_codes, parent_student_links
- **커뮤니케이션 (8)**: announcements, announcement_reads, direct_messages, activity_feed, feed_reactions, study_groups, study_group_members, study_group_messages

| **프레임워크** | Next.js 16 App Router + Turbopack | Server Components로 초기 로딩 최소화, Server Actions로 API Routes 없이 백엔드 로직, Turbopack 빠른 빌드 |
| **언어** | TypeScript strict | 대규모 프로젝트 안정성, AI 호출 결과 타입 안정성 |
| **스타일** | Tailwind CSS 4 + shadcn/ui | 빠른 UI 개발 + 일관된 디자인 시스템 |
| **AI SDK** | Vercel AI SDK v6 | `generateObject`로 구조화 출력, Zod 통합, 스트리밍 지원 |
| **AI 모델** | Claude Sonnet 4.6 | 한국어 품질 최상, 긴 컨텍스트(1M 토큰), 구조화 출력 정확도 |
| **임베딩** | OpenAI text-embedding-3-small | 가장 저렴($0.02/1M), 품질 충분 |
| **STT** | OpenAI Whisper | 한국어 인식률, 다국어 지원 |
| **DB** | Supabase PostgreSQL + pgvector | Auth/DB/벡터검색 일체화, Realtime, Storage, RLS로 보안 |
| **시각화** | D3.js force simulation | 스킬트리 물리 기반 레이아웃, 드래그/줌 커스터마이징 자유도 |
| **차트** | Recharts | React 친화적, 대시보드 차트 충분 |
| **배포** | Vercel | Next.js 최적 배포 환경, Edge Functions, 자동 CI/CD |

## 11. 향후 확장 방향

### 11.1 단기 (1-3개월)
- **카카오 로그인 연동** — 한국 사용자 진입 장벽 제거
- **모바일 앱 (PWA 또는 React Native)** — 학생의 일상 접근성
- **음성 튜터 (TTS)** — ElevenLabs 또는 OpenAI TTS로 튜터 답변 음성 지원
- **실시간 협업 편집** — 교사 간 스킬트리 공동 편집 (Supabase Realtime)

### 11.2 중기 (3-6개월)
- **다국어 지원** — 영어/일본어/중국어 (i18n)
- **교사 커뮤니티** — 공개 스킬트리 마켓플레이스 (교사가 만든 스킬트리 공유)
- **학부모 모바일 알림** — 위험 신호 즉시 푸시 알림
- **고급 분석 대시보드** — 학교 단위 AI 인사이트 리포트

### 11.3 장기 (6-12개월)
- **B2B 학원/학교 라이선스** — SaaS 모델
- **교과서 파트너십** — 출판사와 공식 스킬트리 배포
- **AI 튜터 음성 대화** — ElevenLabs Conversational AI 완전 통합
- **학습 결과 예측 모델** — ML 기반 성적 예측 + 조기 개입
- **VR/AR 확장** — 3D 공간에서 스킬트리 탐험
